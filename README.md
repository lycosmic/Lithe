# Lithe (轻阅)

> "轻盈，敏捷，优雅。"

一个为大学毕业设计而构建的 Android 电子书阅读器。

此项目的核心哲学是 **"Lithe"** —— 打造一款**简单**、**简洁**、**快速**的阅读应用。它追求的是工程上的极简主义，而不是功能的堆砌。

---

## 核心理念 (Core Philosophy)

* **⚡ 快速 (Fast):** 追求极速的应用启动、流畅的翻页性能和最低的内存占用。
* **🍃 简洁 (Concise):** 采用极简主义 (Minimalist) UI 设计，提供无干扰的沉浸式阅读体验。
* **🎯 简单 (Simple):** 只关注“阅读”这一核心功能，保持交互逻辑的直观与简单。

## 技术栈 (Tech Stack)

本项目完全基于现代 Android 技术栈构建：

* **UI:** 100% [Jetpack Compose](https://developer.android.com/jetpack/compose) (声明式 UI)
* **语言:** 100% [Kotlin](https://kotlinlang.org/)
* **架构:** MVVM (ViewModel + Repository)
* **异步:** Kotlin Coroutines & Flow (用于 I/O 和 UI 状态管理)
* **偏好设置:** [Proto DataStore](https://developer.android.com/topic/libraries/architecture/datastore) (取代 SharedPreferences)
* **元数据缓存:** [Room](https://developer.android.com/jetpack/androidx/releases/room) (用于书架和阅读进度)
